Magic Words

Say it once,
type it forever.

Spoken triggers expand into snippets, commands and canned text during cleanup, so they land naturally in the sentence. Prefix matches append whatever you say after the trigger. Built-in app actions launch Codex, Claude or ChatGPT by voice.

Memory & Knowledge Graph

Everything you said becomes something you know.

Every dictation joins a local SQLite corpus with full-text search and on-device embeddings. People, projects, tools and ideas link into a living graph. Ask Memory answers questions over your own history with hybrid retrieval and cited sources, routed through a local LLM if you want.

Ask Memory: "What did I decide about the v0.6 release?" → answered from your own transcripts, with sources.
Notes & Scratchpad

A place for the longer thoughts.

Dictate straight into a rich-text notes workspace, or pop out a detachable floating editor that stays on top while you work. Voice notes are stored locally and searchable alongside everything else.

The full spec

Everything in the box.

Pure Swift and SwiftUI, no Electron. A menu-bar app with no Dock icon by design.

01Core dictationPush-to-talk with fn (Right Option fallback), global hotkey capture system-wide, keystroke injection into any focused field, microphone probing and health checks.
02TranscriptionWhisper via Groq for fast English, OpenAI for multilingual and Hinglish, a streaming realtime path, local model support, and a hallucination guard that strips Whisper's known ghost phrases.
03Transformer profilesLLM cleanup per workflow: standard, Developer mode, Agentic developer mode for AI-coding prompts, Prompt Engineer mode, variable-name recognition, and a custom vocabulary that biases output toward your terms.
04The Dynamic IslandA morphing notch pill with live waveform, thinking dots, done tick, hands-free pulse, error states, and a hover panel across transcriptions, notes, memory and stats.
05Floating chipA draggable frosted-glass widget that persists its position, with recording, processing, done, hands-free and contextual warning states.
06Magic WordsSpoken command aliases with editable triggers, prefix matching, snippet expansion, and built-in app actions, all applied during cleanup.
07Context captureKnows what you were doing: frontmost app and window, selected text via Accessibility, optional window screenshot for context-aware cleanup. Privacy-gated, opt-in by default.
08MemoryLocal knowledge graph + Ask Memory chat over your transcripts: FTS5 search, on-device embeddings, hybrid retrieval with cited sources, local LLM routing via LM Studio or Ollama.
09Run LogA local audit trail for every dictation: audio, raw STT, prompts, final text, model, latency and errors, with playback.
10NotesRich-text notes workspace, a detachable floating editor, and stored voice notes, all local.
11InsightsUsage dashboard: lifetime stats, streaks, app breakdown, and an AI-inferred work profile.
12OnboardingGuided permission flow for Microphone, Accessibility and Input Monitoring, with runtime permission state tracking.
13Open sourceMIT licensed. Homebrew cask, DMG releases, and a signed-and-notarized release pipeline.
